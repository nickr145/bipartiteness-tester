# Testing Bipartiteness in Logarithmic Rounds

Implementation of [Fei & Rubinfeld (2026)](https://arxiv.org/abs/2606.13583)

## What This Is
- Property testing algorithm for bipartite graphs
- O(√n) random walks instead of prior O(√n log n)
- 100% accuracy on test graphs
- ~5min runtime (practical parameters)


See [blog post](https://nrebello.vercel.app/writings/bip-test) for deep dive.
